Heaps & Priority Queues

Min/Max heap structures, dynamic tracking of extreme elements, and continuous K-way merges.

💡 Core Mechanics

  • A Heap is a Complete Binary Tree mapped internally to a flat array layout to maintain fast cache locality.
  • Time Complexities: Accessing the Min/Max element is O(1). Inserting an element or extracting the root takes O(log N). Transforming an unsorted array into a heap (Heapify) runs in O(N).
  • Visual Cue: Reach for a Heap whenever a problem asks you to repeatedly track the "top K", "K-th largest/smallest", or continuously fetch the minimum/maximum element from a dynamic stream of incoming data.

1. Unified Heap Implementation (Min & Max)

Medium

Implement a flexible Heap class in JavaScript that supports both Min-Heap and Max-Heap variants using a constructor flag. The class manages structural invariants dynamically through array index arithmetic.

The "Aha!" Moment: Instead of duplicating the entire codebase for Min and Max layouts, wrap the comparison operations in a helper method or check a configuration variable. The underlying swap mechanics, parent/child calculations, and array restructuring remain completely identical.

// Visual Trace Loop / Whiteboard Diagram

class Heap {
    constructor(type = 'min') {
        this.heap = [];
        this.type = type; // 'min' or 'max'
    }

    getLeftChildIndex(i) {
        return 2 * i + 1;
    }

    getRightChildIndex(i) {
        return 2 * i + 2;
    }

    getParentIndex(i) {
        return Math.floor((i - 1) / 2);
    }

    // Helper to abstract the comparison logic based on the heap type
    compare(a, b) {
        // Min-Heap: parent should be smaller than child
        // Max-Heap: parent should be larger than child
        return this.type === 'min' ? a < b : a > b;
    }

    heapifyUp(idx) {
        let i = idx;

        while (i > 0) {
            const parentIdx = this.getParentIndex(i);

            // Compare child with parent based on Min/Max rules
            if (this.compare(this.heap[i], this.heap[parentIdx])) {
                [this.heap[i], this.heap[parentIdx]] = 
                    [this.heap[parentIdx], this.heap[i]];
                i = parentIdx;
            } else {
                break;
            }
        }
    }

    heapifyDown() {
        let i = 0;

        while (true) {
            const leftIdx = this.getLeftChildIndex(i);
            const rightIdx = this.getRightChildIndex(i);
            let targetIdx = i; // Tracks smallest (MinHeap) or largest (MaxHeap)

            // Check if left child breaks the invariant
            if (
                leftIdx < this.heap.length &&
                this.compare(this.heap[leftIdx], this.heap[targetIdx])
            ) {
                targetIdx = leftIdx;
            }

            // Check if right child breaks the invariant
            if (
                rightIdx < this.heap.length &&
                this.compare(this.heap[rightIdx], this.heap[targetIdx])
            ) {
                targetIdx = rightIdx;
            }

            if (targetIdx !== i) {
                [this.heap[i], this.heap[targetIdx]] = 
                    [this.heap[targetIdx], this.heap[i]];
                i = targetIdx;
            } else {
                break;
            }
        }
    }

    insertNode(val) {
        this.heap.push(val);
        this.heapifyUp(this.heap.length - 1);
        return val;
    }

    extractNode() {
        if (this.heap.length === 0) return null;
        if (this.heap.length === 1) return this.heap.pop();

        const rootVal = this.heap[0]; // Extracted minimum or maximum element
        this.heap[0] = this.heap.pop();
        this.heapifyDown();

        return rootVal;
    }

    peek() {
        return this.heap[0] ?? null;
    }

    size() {
        return this.heap.length;
    }
}
