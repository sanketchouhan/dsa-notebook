Binary Search

Logarithmic interval split algorithm for searching sorted domains.

💡 Core Mechanics

  • The input collection must be sorted to apply traditional Binary Search.
  • Time Complexity: O(log N) because the search space is cut exactly in half each iteration.
  • Space Complexity: O(1) when using an iterative array pointer loop.

704. Binary Search

Easy

Given an array of integers nums which is sorted in ascending order, and an integer target, write a function to search target in nums. If target exists, then return its index. Otherwise, return -1.

The "Aha!" Moment: Calculate the center index using let mid = left + Math.floor((right - left) / 2) instead of Math.floor((left + right) / 2). This avoids potential integer overflow vulnerabilities in large datasets across common runtime engines. 
function search(nums, target) {
    let left = 0;
    let right = nums.length - 1;

    while (left <= right) {
        // Safe middle calculation
        let mid = left + Math.floor((right - left) / 2);

        if (nums[mid] === target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1; // Discard left half
        } else {
            right = mid - 1; // Discard right half
        }
    }

    return -1; // Target not found
}
